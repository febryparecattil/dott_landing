Usage

Do / Don't

Do

  • Use Tangerine as the only accent/CTA color across the product
  • Use dark ink text on Tangerine buttons (4.90:1)
  • Keep Success / Warning / Error reserved for real status states
  • Use Mist and Periwinkle interchangeably for calm backgrounds
  • Run new colors through a contrast checker before shipping

Don't

  • Add a second "loud" accent color alongside Tangerine
  • Use white text on Tangerine, Warning, or any light swatch
  • Reuse Success green or Error red for decoration/illustration
  • Introduce new hues without checking distance from existing ones
  • Rearrange or recolor the logo dot grid
